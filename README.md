# 포토앨범 백엔드 프로젝트

## 프로젝트 선정 동기
백엔드 개발자로써 기본적인 역량을 키우기 위해 유튜버 헬로카일님이 진행하시는 포토앨범 프로젝트를 선정하였습니다.

## 개발 기간
* 2023.03 ~ 2023.04

## 개발 환경
- `Intellij IDEA 2022.3.3`
- `Java 11`
- Spring Boot 2.7.10
- MySQL WorkBench 8.0 CE
- API 테스팅 : Postman
- 문서관리 : Notion
